# AI Knowledge Assistant

This project implements a simple AI knowledge assistant based on 
Retrieval-Augmented Generation (RAG) and AI Agent architecture.

## Features

- AI question answering system
- Knowledge retrieval based on simple RAG pipeline
- AI Agent for automated response generation

## Tech Stack

- Python
- LangChain
- FAISS
- OpenAI API

## Project Structure


ai-knowledge-assistant
│
├── main.py # Main application
├── rag.py # Retrieval module
├── agent.py # Agent logic
├── requirements.txt
└── README.md


## Example

User input:


What is AI?


Output:


Artificial Intelligence is a field of computer science.

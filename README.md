# AI Knowledge Assistant (RAG + AI Agent)

This project implements a simple AI knowledge assistant based on
Retrieval-Augmented Generation (RAG) and AI Agent architecture.

## Features

• Document-based question answering  
• Vector database retrieval  
• RAG pipeline  
• AI Agent reasoning  

## Tech Stack

Python  
LangChain  
FAISS  
OpenAI API  

## Architecture

User Question
     ↓
Vector Search (FAISS)
     ↓
Retrieve Documents
     ↓
LLM Generation
     ↓
Final Answer
